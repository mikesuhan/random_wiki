Bilişimde  Java Secure Socket Extension (JSSE)  ( Türkçe si Java Emin Soket Uzantısı), emin  İnternet  iletişimi sağlayan bir paket seti sağlar.  Secure Sockets Layer  (SSL) (Emin Soket Katmanı) ve  Transport Layer Security  (TLS) (Aktarım Katmanı Emniyeti)  protokol lerinin  Java  teknolojisi sürümünü uygular. Veri  şifreleme , [1]  sunucu kimlik doğrulama, mesaj bütünlüğü ve isteğe bağlı istemci kimlik doğrulaması için fonksiyonellik ihtiva eder.
